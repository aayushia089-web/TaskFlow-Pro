# 📖 About

**TaskFlow Pro** is a full-stack Task Management System developed using **Spring Boot**, **MongoDB**, **HTML**, **CSS**, and **JavaScript**.

It enables users to create, update, delete, search, and organize tasks efficiently through a premium dashboard with modern animations and responsive design.

# 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/aayushia089-web/TaskFlowPro.git
```

### Open Project

Import the project into **Spring Tool Suite (STS)** or **IntelliJ IDEA**.

### Configure MongoDB

```properties
spring.data.mongodb.uri=mongodb://localhost:27017/taskflowpro
```

### Run

Start MongoDB and run:

```
TaskFlowProApplication.java
```

Visit:

```
http://localhost:8080
```
